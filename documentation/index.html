Main Theme Settings

Menu Bar opens over page or pushes page

Add "menu-bar-ontop" class to "body" tag to make it opens over page

Container width can be set to 960px (by default it's 1200px)

Add "cont-960" class to "body" tag to make container 960px wide

Image Gallery

Images width and height in galleries don't matter. Just make sure all images sizes in gallery has the same proportion. Different images proportions would make gaps between images.

"Load More" button

For demo purpose we call more gallery items from ajax/*.html files accordingly.

In main.js file you assets/js/main.js file find this code:

      $.ajax({
         type: "POST",
         url: 'ajax/gallery.html',

where url path replace it to your own server side script path which will call data from db instead html files.

* Please note:

When you open html file, load more button will not work, becaue "ajax/gallery.html" is relative url path.
So if you want to ensure it works, simply upload this template to your hosting, or run in your local http server thru url.
In that case it will work.

SASS Files

All SASS files are located under scss directory

  • bootstrap - bootstrap framework
  • vendor - all vendor styles
  • _common.scss - common styles and elements
  • _components.scss - all template components
  • _layout.scss - template pages layouts
  • _menu-bar.scss - Menu Bar styles
  • _settings.scss - settings file (you can change all theme settings here)

 

JS Files

You can find all JS files under assets/js directory

  • bootstrap - Bootsprap Framework JS files
  • vendor - all defferent libraries and small plugins
  • main.js - main JS files for initializing scripts
  • plugins.js  - template's plugins

PHP FILES

There is only one PHP file - formmail.php
This file responsable for contact form sending. Your server should support sendmail function in order the script can send email.
